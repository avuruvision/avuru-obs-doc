# API

The **`hub`** exposes a client-agnostic API contract used by the UI and any
future client (CLI, Grafana data source).

- **REST / WebSocket** — query signals, stream updates.
- **OpAMP** — remote configuration for agents and collectors.
- **Protobuf contracts** — shared across Rust ↔ Go ↔ TypeScript (`proto/` in the
  engine repo).
