# Configuration

avuru obs is configured through Helm **values** (Kubernetes) and environment
variables (gateway on VMs).

```yaml
# values.yaml (excerpt)
gateway:
  otlp:
    http: 4318
    grpc: 4317
clickhouse:
  persistence:
    size: 50Gi
sensor:
  ebpf:
    enabled: true
```
