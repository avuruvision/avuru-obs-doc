# Kubernetes

On Kubernetes, the `sensor` DaemonSet enriches every signal with pod metadata
(`k8s.namespace.name`, `k8s.pod.name`, `k8s.node.name`) so traces, metrics and
logs are all attributable to the workload that produced them.

See [Setup → Kubernetes](/docs/setup/kubernetes) for the install.
