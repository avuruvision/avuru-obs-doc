Welcome to the **avuru obs** documentation site.

avuru obs is all-in-one observability — traces, metrics, logs and continuous
profiling — in a single install, on one storage engine, with **zero code
changes**. eBPF discovers your services; OpenTelemetry carries the detail.

{/* truncate */}

This site is where the onboarding funnel, guides, reference and integrations
live. The fastest way in is the
[30-second guide](/docs/getting-started/30-seconds): one `helm install` and the
service map lights up.

The docs are open and bilingual (English + French). Found a typo or a gap? Hit
**"Edit this page"** on any doc — it's a three-click pull request from there.
